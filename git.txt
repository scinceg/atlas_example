git init - инициализируем гит в нашем проекте (создаем файл гита в нашем проекте)
создаем новый файл
git add -A - добавялем все файлы в index (промежуточное звено между проектом и файла гита)
git commit -m "add index.html" - сохраняем изменения в файле гита
далем нескольк коммитов
git reset --hard 702573b2251127950bad405296a75d402a4e4da2
откатывем проект до опредленного состояние
702573b2251127950bad405296a75d402a4e4da2 - id коммита (состояния)

--------------------------Ветки--------------------------------------
git branch - посмотреть все ветки
зеленым цветом выделяется текущая ветка, гдемы с вами находимся
git branch dev_js
dev_js - название новой ветки
фиксируем все изменение в новой ветки
git checkout dev_js (переключпение между ветками)
git megre - объеденить текущую ветку с другой
git merge dev_js - объеденяем текущую ветку (master) с dev_js
